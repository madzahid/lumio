== Description ==

Lumio is a clean, minimal WordPress theme built for high-performance blogs.
It features Syne & Figtree typography, a radial glow hero section, pastel
category cards, and a floating table of contents on single posts. Designed for
speed, readability, and SEO — works great with Rank Math or Yoast, and includes
a built-in SEO fallback for sites without an SEO plugin.

Features:
* Syne & Figtree Google Fonts
* Radial glow hero section
* Pastel category card grid
* Floating table of contents on single posts (auto-generated from headings)
* Social share buttons on single posts
* Custom local avatar support (no plugin required)
* Built-in SEO meta fallback (Open Graph, description)
* Primary and footer navigation menus
* Threaded comments support
* Featured image support
* Translation-ready (text domain: lumio)

== Installation ==

1. Download the theme .zip file.
2. Go to Appearance > Themes > Add New > Upload Theme.
3. Upload the lumio.zip and click Install Now.
4. Activate the theme.
5. Go to Appearance > Menus to set up your Primary and Footer menus.
6. Set your site tagline under Settings > General — it appears in the hero section.

== Frequently Asked Questions ==

= Does this theme require any plugins? =
No. The theme works standalone. It is compatible with Rank Math and Yoast SEO
and will defer to them if active.

= Where does the hero description come from? =
It pulls from your site tagline — Settings > General > Tagline.

= How do I change the category card colours? =
The pastel colours cycle through four CSS variables (--color-pastel-purple,
--color-pastel-cyan, --color-pastel-yellow, --color-pastel-red) defined in
style.css. You can override them in Appearance > Customize > Additional CSS.
