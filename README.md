## Description
HackMonks is a premium, minimal WordPress theme designed for high-performance blogs. It features:
- **Typography:** Syne & Figtree
- **Design:** Radial glow hero section, pastel card layouts
- **Performance:** Optimized for speed and SEO

## Installation

1.  Download the theme `.zip` file.
2.  Go to your WordPress Dashboard -> Appearance -> Themes.
3.  Click **Add New** -> **Upload Theme**.
4.  Choose the `hackmonks.zip` file and click **Install Now**.
5.  Activate the theme.

## Development

To work on this theme locally:

1.  Clone this repository into your `wp-content/themes/` directory:
    ```bash
    git clone https://github.com/madzahid/hackmonks-theme.git hackmonks
    ```
